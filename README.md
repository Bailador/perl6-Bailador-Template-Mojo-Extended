# NAME

Bailador::Template::Mojo::Extended

# SYNOPSIS

```perl6
use Bailador;
use Bailador::Template::Mojo::Extended;

renderer Bailador::Template::Mojo::Extended.new;

```

# DESCRIPTION

This module adds extra features to the default ::Mojo renderer.

# METHODS

## `.new`

```perl6

```
